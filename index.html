  • Clarify: augmenting existing or creating new?
  • Assess maturity: data, talent, infra, governance

"$50M spent on AI, limited ROI. Fix it."

  • Audit: which initiatives deliver, which don't
  • Root cause: data? wrong problems? no prod path?
  • Kill underperformers ‚Äî sunk cost discipline
  • Redirect to highest-ROI initiatives
  • Implement stage-gate for future investments

  • Accelerate via partnerships/acquisitions
